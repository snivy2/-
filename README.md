### Customize configuration

一个用于旅游、酒店行业的价格设置日历组件；

由于对接数据较特殊，所以只上传了单机的部分----也就是可以设置价格，但是具体实战中还需要自己根据接口返回的日期数据进行循环匹配；

不过代码也不厚重 主题部分几百行 代码水平也很幼稚，基本上都能看得懂，具体效果大概如下图
